# The Software Syndicate WordPress Theme

A modern, dark-themed WordPress theme designed for "The Software Syndicate" coding community. This theme allows clients to easily edit content while maintaining consistent styling and branding.

## Features

- **Modern Dark Design**: Clean, professional dark theme with cyan accent colors
- **Easy Content Management**: Clients can edit pages through WordPress admin without touching code
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Customizable Hero Section**: Edit homepage hero content through WordPress Customizer
- **Dynamic Navigation**: Easy menu management through WordPress menu system
- **Widget Support**: Sidebar and footer widget areas for flexible content
- **SEO Friendly**: Proper semantic HTML5 structure
- **Social Media Integration**: Footer social media links
- **Comment System**: Full comment support for blog posts

## Installation

1. Upload the `community` folder to `/wp-content/themes/` directory
2. Activate the theme through WordPress admin → Appearance → Themes
3. Configure menus through Appearance → Menus
4. Customize hero section through Appearance → Customize → Hero Section

## Customization Options

### Hero Section
Navigate to **Appearance → Customize → Hero Section** to modify:
- Hero title
- Hero subtitle
- Primary button text and URL
- Secondary button text and URL

### Navigation
Set up menus through **Appearance → Menus**:
- Primary Menu: Main navigation
- Footer Menu: Footer links

### Widgets
Add widgets through **Appearance → Widgets**:
- Primary Sidebar: Appears on blog pages
- Footer Widget Area: Appears in footer
